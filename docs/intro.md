## Key Features

| Feature | Description |
|---------|-------------|
| **Contact Management** | Store names, multiple phone numbers, emails, addresses, and more |
| **Circle Categories** | Organize contacts by relationship: Family, Friends, Work, Sport, Neighbors |
| **Import Contacts** | Import from VCF files, CSV, or directly from Google Contacts |
| **Activity Tracking** | Log outreach attempts and meetings with detailed notes |
| **Team Hierarchy** | View your team members and their sub-teams |
| **Multi-language** | Available in English and Slovak |
